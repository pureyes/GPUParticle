# Particle Texture Setup

This directory should contain particle textures for the GPU particle system.

## Recommended Texture:

Create a simple circular gradient texture in Unity:
1. Right-click in this folder
2. Create > 2D > Sprite
3. Draw a white circle with soft edges
4. Or import a PNG file with a circular gradient

Alternative: Use Unity's built-in "Default-Particle" texture or create a simple white texture.

## Texture Settings:
- Texture Type: Sprite (2D and UI) or Texture
- Filter Mode: Bilinear
- Wrap Mode: Clamp
- Compression: Normal Quality
